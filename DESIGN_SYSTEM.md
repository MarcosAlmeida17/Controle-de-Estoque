# DESIGN_SYSTEM.md

Este arquivo documenta o padrão visual da aplicação.

## Princípios

- Interface administrativa simples.
- Português do Brasil.
- Compatível com tema claro e escuro.
- Uso obrigatório de Heroicons.
- Feedback obrigatório via SweetAlert2.

## Paleta

### Cores principais
- Primary: #0EA5E9
- Primary Hover: #0284C7
- Secondary: #475569
- Success: #22C55E
- Warning: #F59E0B
- Danger: #EF4444

### Light Mode
- Background: #F1F5F9
- Surface: #FFFFFF
- Text Primary: #0F172A
- Border: #CBD5E1

### Dark Mode
- Background: #020617
- Surface: #0F172A
- Text Primary: #F8FAFC
- Border: #334155

## Layout

- Sidebar fixa.
- Header superior.
- Conteúdo central.
- Dashboard com cards.
- Formulários centralizados.

## Componentes

### Cards
Indicadores de estoque, movimentações e relatórios.

### Tabelas
- Hover nas linhas.
- Paginação.
- Estado vazio.
- Coluna de ações.

### Formulários
- Labels claros.
- Campos obrigatórios.
- Validação antes do envio.

## Dashboard

Cards mínimos:
- Total de produtos.
- Estoque baixo.
- Entradas hoje.
- Saídas hoje.
- Valor total em estoque.
