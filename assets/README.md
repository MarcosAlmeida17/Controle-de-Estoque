Coloque aqui o logo do sistema com o nome `logo.png`.

- Caminho esperado (usa pela aplicação): `assets/logo.png`
- Para páginas em `pages/`, o caminho usado é `../assets/logo.png`.

Observações:
- O arquivo enviado na conversa deve ser salvo neste local. O arquivo recomendado é PNG ou JPG.
- Se quiser que eu adicione o arquivo diretamente ao repositório, faça upload do arquivo PNG através da interface ou me permita acesso direto ao binário.

Exemplo de comando (Windows PowerShell):

mv "C:\caminho\para\download\logo.png" "assets\logo.png"

Se preferir, posso incorporar o logo inline como data URI no HTML — avise se quiser isso.
